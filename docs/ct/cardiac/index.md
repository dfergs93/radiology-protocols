# ❤️ Cardiac CT Protocols

Protocols for coronary CTA, calcium scoring, and functional cardiac imaging.

---

## Quick Reference


| Clinical Scenario | Key Protocol Features | Heart Rate |
|------------------|----------------------|------------|
| Coronary CTA | Gated, arterial phase | <60 bpm |
| Calcium score | Non-contrast, gated | Any |
| TAVR planning | Non-gated, arterial phase | <70 bpm |
| Bypass grafts | Gated, extended coverage | <65 bpm |


---


## Heart Rate Management

!!! danger "Critical for Image Quality"
    Target HR: <60 bpm for coronary CTA

**Beta Blocker Protocol:**
- Metoprolol 25-100 mg PO (1 hour before)
- IV Metoprolol 5 mg increments (max 15 mg)

**Contraindications:**
- Severe asthma/COPD
- Heart block (2nd/3rd degree)
- SBP <100 mmHg

### Nitroglycerin
- Dose: 0.4-0.8 mg SL
- Timing: 3-5 min before scan
- Contraindications: Recent PDE5 inhibitor, SBP <90

### Contrast Timing
\[
\text{Scan Delay} = \text{Peak Time} + 4-6 \text{ seconds}
\]
